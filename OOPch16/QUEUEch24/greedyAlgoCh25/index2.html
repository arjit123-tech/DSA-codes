CSS Styling Methods

This webpage demonstrates the use of both internal CSS (styles defined within the <style> tag in the HTML document) and external CSS (styles loaded from an external .css file).

Internal CSS is used for page-specific styles that are unique to this page, while external CSS contains reusable styles that could be applied across multiple pages.

Internal CSS

Internal CSS is defined within the <style> element inside the <head> section of an HTML document.

  • Applies only to the current page
  • Useful for page-specific styles
  • Can override external CSS
  • Increases page load time slightly

This card is styled with internal CSS.

External CSS

External CSS is defined in a separate .css file and linked to the HTML document using the <link> tag.

  • Can be reused across multiple pages
  • Easier to maintain and update
  • Better for website performance
  • Promotes consistency across a site

This card uses styles from the external CSS file.

Internal vs External CSS Comparison

Below is a comparison of the characteristics of internal and external CSS:

Feature Internal CSS External CSS
Definition Location Inside the HTML file, within <style> tags In a separate .css file
Scope Applies only to the current page Can apply to multiple pages
Maintenance Harder to maintain across multiple pages Easier to maintain and update
Page Load Speed Slightly increases HTML file size Can be cached by browsers for faster loading
Use Case Page-specific styling Site-wide consistent styling

CSS Priority Order (Highest to Lowest)

  1. Inline Styles (style attribute)
  2. Internal CSS (<style> tag)
  3. External CSS (external .css file)
  4. Browser Default Styles
